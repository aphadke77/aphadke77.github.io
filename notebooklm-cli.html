NotebookLM CLI Cheat Sheet

Unofficial Python CLI for Google NotebookLM — notebooklm-py package

Quick Start Flow:
notebooklm login  → authenticate
notebooklm list  → find notebook ID
notebooklm use <id>  → set active notebook
notebooklm source add "<url>"  → add a source
notebooklm ask "your question"  → chat with notebook
notebooklm artifact list  → see generated artifacts
notebooklm download report  → export artifact
