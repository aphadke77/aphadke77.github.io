VirtualBox CLI Commands Reference

A comprehensive guide to VirtualBox command-line interface (VBoxManage)

VM Management

Creating VMs
Create a new VM
VBoxManage createvm --name "VM_NAME" --ostype "OS_TYPE" --register
Clone a VM
VBoxManage clonevm "SOURCE_VM" --name "CLONE_NAME" --register
Import appliance
VBoxManage import "/path/to/appliance.ova" --vsys 0 --vmname "NEW_VM_NAME"
Modifying VMs
Rename a VM
VBoxManage modifyvm "OLD_VM_NAME" --name "NEW_VM_NAME"
Set memory (MB)
VBoxManage modifyvm "VM_NAME" --memory 4096
Set CPU count
VBoxManage modifyvm "VM_NAME" --cpus 2

VM Control

Starting VMs
Start VM (GUI)
VBoxManage startvm "VM_NAME"
Start VM (headless)
VBoxManage startvm "VM_NAME" --type headless
Stopping VMs
Graceful shutdown
VBoxManage controlvm "VM_NAME" acpipowerbutton
Force power off
VBoxManage controlvm "VM_NAME" poweroff
Save VM state
VBoxManage controlvm "VM_NAME" savestate

VM Information

Listing VMs
List all VMs
VBoxManage list vms
List running VMs
VBoxManage list runningvms
VM Details
Show VM info
VBoxManage showvminfo "VM_NAME"
Check VirtualBox version
VBoxManage --version

Snapshots

Managing Snapshots
Take a snapshot
VBoxManage snapshot "VM_NAME" take "SNAPSHOT_NAME"
Restore a snapshot
VBoxManage snapshot "VM_NAME" restore "SNAPSHOT_NAME"
Delete a snapshot
VBoxManage snapshot "VM_NAME" delete "SNAPSHOT_NAME"
