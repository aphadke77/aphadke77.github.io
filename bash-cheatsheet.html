Bash Command Line Cheatsheet

A comprehensive reference for Bash shell commands and syntax

Basic Commands

CommandDescriptionExample
lsList directory contentsls -la
cdChange directorycd /path/to/directory
pwdPrint working directorypwd
mkdirCreate a new directorymkdir new_directory
touchCreate a new file or update timestamptouch file.txt
cpCopy files or directoriescp file.txt /path/to/copy
mvMove or rename files or directoriesmv file.txt newname.txt
rmRemove files or directoriesrm file.txt or rm -rf directory
catConcatenate and display file contentcat file.txt
lessView file contents with paginationless large_file.txt

File Permissions

CommandDescriptionExample
chmodChange file mode (permissions)chmod 755 script.sh
chownChange file ownerchown user:group file.txt

Permission numbers: 4=read, 2=write, 1=execute

Common sets: 755 (rwxr-xr-x), 644 (rw-r--r--), 777 (rwxrwxrwx)

Text Processing

CommandDescriptionExample
grepSearch for patterns in filesgrep "pattern" file.txt
sedStream editor for filtering and transforming textsed 's/old/new/g' file.txt
awkPattern scanning and processing languageawk '{print $1}' file.txt
sortSort lines of text filessort file.txt
uniqReport or omit repeated linessort file.txt | uniq
wcCount lines, words, and characterswc -l file.txt
cutRemove sections from each linecut -d ',' -f 1 data.csv

Process Management

CommandDescriptionExample
psReport process statusps aux
topDisplay processes in real-timetop
killTerminate processeskill PID or kill -9 PID
bgResume suspended jobs in the backgroundbg %1
fgBring job to the foregroundfg %1
nohupRun command immune to hangupsnohup command &

I/O Redirection

OperatorDescriptionExample
>Redirect output to a file (overwrite)ls > output.txt
>>Redirect output to a file (append)echo "text" >> file.txt
|Pipe output to another commandls -la | grep ".txt"

Useful Shortcuts

ShortcutDescription

Ctrl+ZSuspend current process
Ctrl+DExit current shell
Ctrl+LClear the screen

CommandDescriptionExample
