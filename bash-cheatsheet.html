Bash Command Line Cheatsheet

A comprehensive reference for Bash shell commands and syntax

Basic Commands

Command Description Example
ls List directory contents ls -la (list all files including hidden ones in long format)
cd Change directory cd /path/to/directory
pwd Print working directory pwd
mkdir Create a new directory mkdir new_directory
touch Create a new file or update timestamp touch file.txt
cp Copy files or directories cp file.txt /path/to/copy
mv Move or rename files or directories mv file.txt newname.txt
rm Remove files or directories rm file.txt or rm -rf directory
cat Concatenate and display file content cat file.txt
less View file contents with pagination less large_file.txt
Be very careful with rm -rf as it forcefully removes directories and their contents without confirmation.

File Permissions

Command Description Example
chmod Change file mode (permissions) chmod 755 script.sh or chmod u+x script.sh
chown Change file owner chown user:group file.txt

Permission numbers:

  • 4 = read (r)
  • 2 = write (w)
  • 1 = execute (x)

Common permission sets:

  • 755 = rwxr-xr-x (owner: rwx, group: r-x, others: r-x)
  • 644 = rw-r--r-- (owner: rw-, group: r--, others: r--)
  • 777 = rwxrwxrwx (everyone has full permissions - use with caution)

Text Processing

Command Description Example
grep Search for patterns in files grep "pattern" file.txt or grep -r "pattern" directory/
sed Stream editor for filtering and transforming text sed 's/old/new/g' file.txt
awk Pattern scanning and processing language awk '{print $1}' file.txt
sort Sort lines of text files sort file.txt or sort -n numbers.txt
uniq Report or omit repeated lines sort file.txt | uniq
wc Count lines, words, and characters wc -l file.txt (count lines)
cut Remove sections from each line cut -d ',' -f 1 data.csv

Process Management

Command Description Example
ps Report process status ps aux
top Display processes in real-time top
kill Terminate processes kill PID or kill -9 PID (force kill)
bg Resume suspended jobs in the background bg %1
fg Bring job to the foreground fg %1
jobs List background jobs jobs
nohup Run command immune to hangups nohup command &

I/O Redirection

Operator Description Example
> Redirect output to a file (overwrite) ls > output.txt
>> Redirect output to a file (append) echo "text" >> file.txt
< Redirect input from a file sort < input.txt
2> Redirect error output command 2> error.log
&> Redirect both standard output and errors command &> output.log
| Pipe output to another command ls -la | grep ".txt"

Bash Script Basics

Script Header

#!/bin/bash
# Script description

Variables

# Define variables (no spaces around =)
NAME="John"
AGE=30

# Using variables
echo "Hello, $NAME"
echo "You are $AGE years old"

Conditionals

if [ "$NAME" == "John" ]; then
    echo "Hello John!"
elif [ "$NAME" == "Jane" ]; then
    echo "Hello Jane!"
else
    echo "Hello stranger!"
fi

Loops

# For loop
for i in {1..5}; do
    echo "Number: $i"
done

# While loop
count=1
while [ $count -le 5 ]; do
    echo "Count: $count"
    ((count++))
done

Functions

function greet() {
    local name=$1
    echo "Hello, $name!"
}

# Call function
greet "World"

Save your script with a .sh extension, make it executable with chmod +x script.sh, and run it with ./script.sh

Networking

Command Description Example
ping Send ICMP ECHO requests to network hosts ping google.com
traceroute Print the route packets trace to a network host traceroute google.com
netstat Network statistics netstat -tuln
ssh Secure shell remote login ssh user@hostname
scp Secure copy (remote file copy) scp file.txt user@hostname:/path
wget Retrieve files from the web wget https://example.com/file.txt
curl Transfer data from or to a server curl -O https://example.com/file.txt
ip Show/manipulate routing, network devices, etc. ip addr show

Compression & Archives

Command Description Example
tar Tape archiver tar -czvf archive.tar.gz directory/
gzip Compress or expand files gzip file.txt
gunzip Uncompress gzipped files gunzip file.txt.gz
zip Package and compress files zip -r archive.zip directory/
unzip List, test and extract compressed files unzip archive.zip

Common tar options:

  • c - create a new archive
  • x - extract files from an archive
  • t - list the contents of an archive
  • v - verbose output
  • f - use archive file
  • z - filter the archive through gzip
  • j - filter the archive through bzip2

Find & Search

Command Description Example
find Search for files in a directory hierarchy find /path -name "*.txt"
locate Find files by name (uses database) locate filename
grep Print lines matching a pattern grep -r "text" /path
