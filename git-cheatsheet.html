Git Command Cheat Sheet

Setting Up

Command Description Example
git init Initialize a new Git repository git init
git clone Clone a repository from a remote source git clone https://github.com/user/repo.git
git config Configure Git settings git config --global user.name "Your Name"
git config --global user.email "your.email@example.com"

Basic Commands

Command Description Example
git status Show working tree status git status
git add Add file contents to the index git add file.txt
git add .
git add -A
git commit Record changes to the repository git commit -m "Commit message"
git commit -am "Add and commit message"
git diff Show changes between commits, commit and working tree, etc. git diff
git diff --staged
git rm Remove files from the working tree and index git rm file.txt
git rm --cached file.txt
git mv Move or rename a file, directory, or symlink git mv old_name.txt new_name.txt

Branching and Merging

Command Description Example
git branch List, create, or delete branches git branch
git branch new-branch
git branch -d branch-name
git checkout Switch branches or restore working tree files git checkout branch-name
git checkout -b new-branch
git checkout -- file.txt
git switch Switch branches (newer alternative to checkout) git switch branch-name
git switch -c new-branch
git merge Join two or more development histories together git merge branch-name
git rebase Reapply commits on top of another base git rebase main
git rebase -i HEAD~3
git cherry-pick Apply changes from specific commits git cherry-pick commit-hash

Remote Repositories

Command Description Example
git remote Manage remote repositories git remote -v
git remote add origin https://github.com/user/repo.git
git remote remove origin
git fetch Download objects and refs from another repository git fetch origin
git fetch --all
git pull Fetch from and integrate with another repository or branch git pull origin main
git pull --rebase origin main
git push Update remote refs along with associated objects git push origin main
git push -u origin main
git push --force

Inspection & Comparison

Command Description Example
git log Show commit logs git log
git log --oneline
git log --graph --pretty=format:'%C(auto)%h%d %s %C(green)(%cr) %C(blue)<%an>'
git show Show various types of objects git show commit-hash
git show HEAD
git blame Show who changed which lines in a file and when git blame file.txt
git tag Create, list, delete, or verify tags git tag
git tag -a v1.0.0 -m "Version 1.0.0"
git push origin --tags

Undoing Changes

Command Description Example
git reset Reset current HEAD to the specified state git reset HEAD file.txt
git reset --soft HEAD~1
git reset --hard HEAD~1
git restore Restore working tree files (newer alternative) git restore file.txt
git restore --staged file.txt
git revert Create a commit that undoes changes from a previous commit git revert commit-hash
git clean Remove untracked files from the working tree git clean -n
git clean -f
git stash Stash changes in a dirty working directory git stash
git stash pop
git stash list
git stash drop

Advanced Operations

Command Description Example
git bisect Use binary search to find a commit that introduced a bug git bisect start
git bisect bad
git bisect good commit-hash
git reflog Manage reference logs; helpful for recovering lost commits git reflog
git submodule Initialize, update, or inspect submodules git submodule add https://github.com/user/repo
git submodule update --init --recursive
git worktree Manage multiple working trees attached to the same repository git worktree add ../new-tree branch-name
git filter-branch Rewrite branches (use with caution) git filter-branch --tree-filter 'rm -f passwords.txt' HEAD

Configuration & Aliases

git config --global alias.st status

Common Workflows

Making Changes and Committing

# Start work on a new feature
git checkout -b feature-branch

# Make changes, then stage them
git add changed_files

# Commit with a descriptive message
git commit -m "Add feature XYZ"

# Push to remote repository
git push -u origin feature-branch

Merging a Feature Branch

# Switch to the main branch
git checkout main

# Update from remote
git pull origin main

# Merge your feature branch
git merge feature-branch

# Push the merged changes
git push origin main

Resolving Merge Conflicts

# After a merge conflict occurs:
# 1. Open conflicted files and resolve manually
# 2. Stage the resolved files
git add resolved_files

# 3. Complete the merge
git commit

# 4. Push the merged changes
git push origin main

Updating a Local Branch with Remote Changes

# Fetch remote changes
git fetch origin

# Incorporate changes into current branch
git pull origin current-branch
# OR (if you prefer rebasing)
git pull --rebase origin current-branch
