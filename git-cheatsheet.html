Git Command Cheat Sheet

Setting Up

CommandDescriptionExample
git initInitialize a new Git repositorygit init
git cloneClone a repository from a remote sourcegit clone https://github.com/user/repo.git
git configConfigure Git settingsgit config --global user.name "Your Name"

Basic Commands

CommandDescriptionExample
git statusShow working tree statusgit status
git addAdd file contents to the indexgit add .
git commitRecord changes to the repositorygit commit -m "message"
git diffShow changes between commitsgit diff
git rmRemove files from the working treegit rm file.txt

Branching and Merging

CommandDescriptionExample
git branchList, create, or delete branchesgit branch new-branch
git checkoutSwitch branches or restore filesgit checkout -b new-branch
git mergeJoin two or more development historiesgit merge branch-name
git rebaseReapply commits on top of another basegit rebase main

Remote Repositories

CommandDescriptionExample
git remoteManage remote repositoriesgit remote -v
git fetchDownload objects and refsgit fetch origin
git pullFetch and integrate with another repositorygit pull origin main
git pushUpdate remote refsgit push origin main

Inspection & Comparison

CommandDescriptionExample
git logShow commit logsgit log --oneline
git showShow various types of objectsgit show commit-hash
git tagCreate, list, delete tagsgit tag -a v1.0.0 -m "Version 1"

Undoing Changes

CommandDescriptionExample
git resetReset current HEADgit reset --soft HEAD~1
git restoreRestore working tree filesgit restore file.txt
git revertCreate a commit that undoes changesgit revert commit-hash
git stashStash changes in a dirty working directorygit stash
